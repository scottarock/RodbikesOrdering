The problem

For many small retail businesses, standard ordering tools are integrated with point-of-sale (POS) systems. These systems are often automated to the point where flexibility is lost. For example, when retail businesses run out of seasonal items in their inventory, their POS system may automatically add the items to an order list. Rather than reorder items that are no longer in season, staff may want to place the items on sale and deplete their inventory to make way for new seasonally appropriate items. In addition, many POS systems require that staff enter every item that is in stock into the system, which can create a large amount of data overhead.

The solution: The Want List app

The Want List app provides a more flexible and simpler way to add the items that you need to order. It's easy for new staff to use and can cover special-order items as well.

How to use the Want List app

  • To add, edit, and view items, use the Want List tab.
  • To place orders, use the Place an Order tab. Note the items on the order may be edited during ordering.
  • To search all items, use the Items tab.
  • To view orders that have been placed, use the Orders tab
  • To add, edit, and view vendors, use the Vendors tab.
